# Fruits App

A fully responsive fruits selling e-commerce website built using React and Tailwind CSS. This project focuses on clean UI, fast performance, and a smooth user experience.

This project follows a modern frontend structure and shows how to build a real-world shopping interface step by step.

## Features

- Responsive design for mobile, tablet, desktop

- Modern UI using Tailwind CSS

- Product listing with clean layout

- Add to cart functionality

- Smooth navigation and user flow

- Reusable React components

- Optimized performance

## Tech Stack

- React JS

- Tailwind CSS

- JavaScript

## Installation

Clone the repository

```bash
git clone https://github.com/Nisheshchy/Fruits-App.git

```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
